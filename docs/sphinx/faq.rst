1. I can't install things using ``sdss_install``

   Check to make sure you have the ``SDSS_GITHUB_KEY``, ``SDSS_INSTALL_GRAPHQL_DIR``, and
   ``SDSS_INSTALL_PRODUCT_ROOT`` environment variables set.
